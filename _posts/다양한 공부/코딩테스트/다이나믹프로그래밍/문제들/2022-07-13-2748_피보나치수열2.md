# 1. 문제
문제는 [링크](https://www.acmicpc.net/problem/2748)에 들어가면 있다.

# 2. 정답 코드

문제의 내 정답 코드는 다음과 같다.

```python
N = int(input())
dp_table = [0]*(N+1)
dp_table[0] = 0
dp_table[1] = 1
for i in range(2,N+1):
    dp_table[i] = dp_table[i-1] + dp_table[i-2]

print(dp_table[-1])
```

기본적인 문제라 넘어가자.
