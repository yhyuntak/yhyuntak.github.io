# 1. 정답 코드

문제의 내 정답 코드는 다음과 같다.

```python
n = int(input())

d = [0] * (n+1)

d[1] = 1
d[2] = 2


for i in range(3,n+1):
    d[i] = (d[i-1] + d[i-2]) % 10007

print(d[n])
```
