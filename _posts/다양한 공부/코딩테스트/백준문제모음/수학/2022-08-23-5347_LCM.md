# 1. 문제
문제는 [링크](https://www.acmicpc.net/problem/5347)에 들어가면 있다.

# 2. 정답 코드

문제의 내 정답 코드는 다음과 같다.

```python
import sys
read = sys.stdin.readline

n = int(read())

for _ in range(n):
    a,b = map(int,read().split())
    aa,bb = a,b

    while a % b != 0 :
        a,b = b, a%b

    print(aa*bb//b)


```

## 문제 풀이

최대 공약수를 구하는 문제를 코드를 응용해서 최소 공배수를 이끌어내자.
