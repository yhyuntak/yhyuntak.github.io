# 1. 문제
문제는 [링크](https://www.acmicpc.net/problem/1110)에 들어가면 있다.

# 2. 정답 코드

문제의 내 정답 코드는 다음과 같다.

```python

N = input()
cycle = 0

original_N = N

while True :


    if len(N) < 2 : N = '0'+N
    temp_sum = str(int(N[0]) + int(N[1]))
    new_num = N[-1] + temp_sum[-1]
    N = new_num
    cycle+=1
    if int(original_N) == int(N):  break

print(cycle)

```
