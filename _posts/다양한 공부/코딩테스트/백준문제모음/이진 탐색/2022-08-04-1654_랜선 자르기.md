# 1. 문제
문제는 [링크](https://www.acmicpc.net/problem/1654)에 들어가면 있다.

# 2. 정답 코드

문제의 내 정답 코드는 다음과 같다.

```python
import sys

read = sys.stdin.readline

K,N = map(int,read().split())
array = []
for _ in range(K):
    array.append(int(read()))

start = 1
end = max(array)

while start<=end :
    mid = (start+end)//2
    temp = 0
    for a in array:
        temp += (a//mid)

    if temp >= N :
        start = mid+1
    else :
        end = mid-1

print(end)



```
