# 1. 문제
문제는 [링크](https://www.acmicpc.net/problem/11053)에 들어가면 있다.

# 2. 정답 코드

문제의 내 정답 코드는 다음과 같다.

```python
import sys

N = int(input())
numbers = list(map(int, sys.stdin.readline().split()))
dp_table = [1] * N
for i in range(1,N):
    for j in range(i):
        if numbers[i] > numbers[j] :
            dp_table[i] = max(dp_table[i],dp_table[j]+1)
print(max(dp_table))


```
